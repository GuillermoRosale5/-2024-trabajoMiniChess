#MiniChess PROJECT

A chess video game developed in C++ with a graphical interface using OpenGL/FreeGLUT. The main program creates the game window and manages keyboard, mouse, and timer events, while the overall logic is organized through an object-oriented structure. The result is an interactive application that combines graphical programming and game logic.

The code is divided into classes that represent the different elements of the game, such as the board, squares, players, and each chess piece. This allows each piece to have its own behavior and the board to control aspects such as valid moves, the game state, and the general rules. Therefore, it is not just a visual representation, but a functional implementation of the game.

One of the most striking features of the project is its setting, as chess is presented with an aesthetic inspired by Star Wars, under the name "Chess Wars." Images, sprites, and music are used to give the game personality, so it is not limited to reproducing classic chess, but seeks to offer a more attractive visual and thematic experience.

In terms of gameplay, the project includes several menus and game modes. It can be played between two people or against the computer, and there's also a special option to activate a gravity mode, which introduces a different mechanic to traditional chess. Important rules such as check, checkmate, pawn promotion, and other checks necessary for the game to function correctly are also implemented.

Overall, it can be said that it's a fairly complete project, because it combines object-oriented programming, graphic design, event handling, and the implementation of game rules. Its goal is not only to recreate chess, but to do so in a more original and dynamic way, adding its own unique atmosphere and some differentiating mechanics.





ESPAÑOL
Trabajo de informática, curso 2023-2024 (minichess), de ingeniería electrónica en la ETSIDI. Este grupo está formado por Javier Alnoso, Guillermo Rosales, Javier García y Daniel van Reenen.

Videojuego de ajedrez desarrollado en C++ con interfaz gráfica mediante OpenGL/FreeGLUT. El programa principal crea la ventana del juego y gestiona los eventos de teclado, ratón y temporizador, mientras que la lógica general se organiza a través de una estructura orientada a objetos. El resultado es una aplicación interactiva en la que se combinan programación gráfica y lógica de juego.

La organización del código está dividida en clases que representan los distintos elementos de la partida, como el tablero, las casillas, los jugadores y cada una de las piezas del ajedrez. Esto permite que cada pieza tenga su propio comportamiento y que el tablero controle aspectos como los movimientos válidos, el estado de la partida y las reglas generales. Por tanto, no se trata solo de una representación visual, sino de una implementación funcional del juego.

Uno de los rasgos más llamativos del proyecto es su ambientación, ya que el ajedrez se presenta con una estética inspirada en Star Wars, bajo el nombre de “Chess Wars”. Se utilizan imágenes, sprites y música para dar personalidad al juego, de forma que no se limita a reproducir el ajedrez clásico, sino que busca ofrecer una experiencia visual y temática más atractiva.

En cuanto a la jugabilidad, el proyecto incluye varios menús y modos de juego. Se puede jugar entre dos personas o contra la máquina, y además existe una opción especial para activar un modo con gravedad, que introduce una mecánica distinta al ajedrez tradicional. También se implementan reglas importantes como el jaque, el jaque mate, la coronación de peones y otras comprobaciones necesarias para que la partida funcione correctamente.

En conjunto, puede decirse que es un proyecto bastante completo, porque combina programación orientada a objetos, diseño gráfico, gestión de eventos e implementación de reglas de juego. Su objetivo no es solo recrear el ajedrez, sino hacerlo de una forma más original y dinámica, añadiendo una ambientación propia y algunas mecánicas diferenciadoras.
